重命名当前分支

提交所有更改的文件，并且生成一个commit

git commit -a会自动将所有已追踪（tracked）文件的更改添加到暂存区并提交，但不会包括新添加的未追踪（untracked）文件。对于新文件需要使用git add .进行添加。

第一次推送

“用于将本地分支推送到远程仓库，并将其与远程仓库中的相应分支建立跟踪关系。这个命令通常在你第一次将某个本地分支推送到远程仓库时使用。” —— LLM

使用本地分支强行替代远程分支

“如果你经常需要进行强制推送，推荐使用 --force-with-lease 选项，这样可以在推送之前检查远程分支是否已经被其他人更新过，从而减少覆盖其他人提交的风险。” —— LLM
